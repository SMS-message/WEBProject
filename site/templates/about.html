О проекте

VideoHoster - бот-сервис с короткими видео с открытым исходным кодом;

Количество строк кода: 374.

Установка

Через installer.sh:

- Запустите файл installer.sh с помощью bash в командной строке, написав команду

bash installer.sh

Ручная установка

- Создать виртуальное окружение:

python -m venv .venv

venv/Scripts/activate

- Установить библиотеки:

pip install -r requirements.txt

- Создать необходимые для работы файлы:

> директория bot/logs > директория bot/videos > файл bot/data/blacklist.txt

Администрирование проекта

Диалоги между ботом и пользователями сохраняются в директорию bot/logs;

При возникновении ошибок, они выводятся в консоль.
